## Si te Instalosh

### Hapi 1: Shkarko
Kliko linkun e shkarkimit me lart per te marre `Takvimi-per-Evrope-Setup.exe`

### Hapi 2: Instalo
1. Hap skedarin qe shkarkove
2. Ndiq udhezimet e instalimit

### Hapi 3: Perdor
Aplikacioni do te shfaqet ne system tray (ikona prane ores). Kliko mbi te per te pare kohet e namazit.

## Vecorite

- **Kohet e namazit** per 58 qytete ne 13 vende Evropiane

- **Ruajtja e qytetit** - mban mend zgjedhjen tende

## Ndertimi nga Kodi

```bash
# Klono
git clone https://github.com/drilonjaha/Takvimi-per-Evrope-Windows.git
cd Takvimi-per-Evrope-Windows

# Instalo dependencies
npm install

# Testo
npm start

# Build per Windows
npm run build:win
```
